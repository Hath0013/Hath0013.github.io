Insights & outcomes

My responsibility as the design lead was to spearhead the in-house design of the magazines and format this content for the magazine websites.

I established a design system, a project timeline and a design process. I designed the layout and typography for all of the advertorial and editorial and approximately 40% of the advertising for each issue.

The skills I learned at CMG would set me up for future implementation of processes and systems. I was able to accomplish these key achievements:

  • Implemented a design process and system. There was no process or system in place for how the in-house magazine design would work as it had not been done before at CMG. This was key in establishing the structure of how I would conduct my work and how I would work with the magazine editors.
  • Established project management. To ensure we would meet each issue deadline, I built a structured project timeline complete with objectives, milestones and deadlines for each issue. Setting up these tools helped me stay on task and identify when I was falling behind.
