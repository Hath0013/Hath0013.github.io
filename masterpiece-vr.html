MasterpieceVR

User Experience & Interface Design

MasterpieceVR is a 3D sculpting and painting tool that allows users to build tangible objects, colourful environments, and high quality models. It also allows you to sculpt with friends in the same room in real time, or invite spectators to watch while you create.

Image Source

On a team of three game developers and two UX designers, we were tasked with designing and building an annotation system for MasterpieceVR. This system allows users to add text to any object in virtual space.

We designed the interface and user experience for the annotation system. We made interactive prototypes in Adobe XD and the developed them in Unity. We also conducted user testing of the prototypes and gathered feedback to improve the system.

We presented our annotation system during ReAction day at Algonquin College. I created whiteboard art (above) and a poster for our showcase.
